🏯 Great Wall of China Project
This is a three-page website about the history and building of the Great Wall of China. It is built using only HTML to show how to 
link pages together and organize information correctly.

  Main Features
1.Three Connected Pages: You can move between the Home, Construction, and Legacy pages using a navigation menu at the top.

2.Interactive "Reveal" Box: On the Construction page, users can click a "secret" box to find out about the sticky rice used 
in the wall.

3.Comparison Table: On the Legacy page, there is a clean table that compares common myths to the actual facts.

4.Helpful Symbols: I used emojis like 🛠️ and 📍 to make the headers and lists more interesting to look at.

  Lessons Learnt
1. Linking is Everything
I learned that using href="filename.html" is how the internet stays connected. If the filename is even one letter wrong, the link
won't work!

2. Choosing the Right Tag
I learned that different tags have different jobs:

a.<ul> and <li> are for simple lists.

b.<dl> and <dt> are for defining specific places or terms.

c.<table> is the best way to compare two different sets of data.


3. Interactive HTML
I discovered that HTML has built-in "magic" like the <details> tag, which lets me hide and show information without needing
complex code.

How to View the Project
Keep all three HTML files (index.html, construction.html, legacy.html) in the same folder.

Open index.html in any web browser.

Use the menu at the top to explore the history of the wall!
